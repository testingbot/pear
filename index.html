TestingBot PEAR channel

Registering the channel: 
pear channel-discover testingbot.github.com/pear
Listing available packages: 
pear remote-list -c testingbot
Installing a package: 
pear install testingbot/package_name
Installing a specific version/stability: 
pear install testingbot/package_name-1.0.0
pear install testingbot/package_name-beta
Receiving updates via a feed: 
http://testingbot.github.com/pear/feed.xml
